== Description ==
This plug-in will remove/hide pro features so the interface will be a bit cleaner.

<b>Keep in mind:</b> Remove the plug-in if you want to use [Elementor Pro](https://be.elementor.com/visit/?bta=207645&brand=elementor)

== Installation ==
* install Elementor (free)
* install the plugin
